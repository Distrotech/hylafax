HYLAFAX CHANGE INFORMATION
    Current Version: v3.0beta113
    Previous Version: v3.0beta112
    Master FTP Site: sgi.com (192.48.153.1), directory sgi/fax

This document describes the changes made to the software between the previous and current versions (see above). If you don't find something listed here, then it was not done in this timeframe, or it was not considered important enough to be mentioned. The following information is located here:

  • Changes in the software configuration
  • Changes in the modem config files
  • Changes to faxgetty
  • Changes to pagesend
  • Changes to faxaddmodem
  • Changes to sendpage

CHANGES IN THE SOFTWARE CONFIGURATION:
  • an installClient target was added to the top level Makefile for installing only client-specific materials
  • an additional check was added to verify a proper C++ runtime environment is present

CHANGES IN THE MODEM CONFIG FILES:
  • new config file for the GVC MaxTech 28800
  • new config file for the Motorola Lifestyle 28.8
  • new config file for RC288DPi-based modems like the and the Microcom DeskPort 28.8S
  • the PPI FXMT config file was cleaned up and tested with the PM28800FXMT
  • several config files for Rockwell-based modems were renamed and cleaned up

CHANGES TO FAXGETTY:
  • the Class 1 driver now properly responds to a T2 timeout after receiving EOM
  • the Class 2/2.0 drivers now hang up more quickly after a successful facsimile receive operation

CHANGES TO PAGESEND:
  • extraneous status messages returned by a modem when dialing are ignored

CHANGES TO FAXADDMODEM:
  • the Class 1 modem identification procedure now uses ATI0 and ATI3 since some modems return the same product code for ATI0 but have AT command set incompatibilities (ATI3 typically returns a product information string)

CHANGES TO SENDPAGE:
  • temporary files are now removed
