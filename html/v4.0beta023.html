This document describes the changes made to the software between the previous and current versions (see above). If you don't find something listed here, then it was not done in this timeframe.

CHANGES:
  • notify(1M) now returns the pagernumber and not the number of the service provider; submitted by aspeer@adam.com.au (Andrew Speer), 12 Nov 1996
  • make "\n" to break lines for faxcover(1); like:
    faxcover -c "Dear blah,\n, here's the fax" ...
    submitted by pthirose@ns.net (Paul Hirose), 4 Dec 1996
  • nuke "\r" at end of line read from config(4) file because "fgets(3C)" reads them into memory; submitted by steve@genie96.com (Steve Williams), 10 Dec 1996
  • fix for a bug in "fprintf(...)" arguments causing a problem if "%z" option for ModemFmt is used; submitted by I.Senior@bom.gov.au (Ian Senior), 11 Dec 1996
  • various fixes for textfmt(1) to make is reliable 8-bit clean on any platform and any OS; submitted by leif@lege.com (Leif Erlingsson), 13 Feb 1997 and others;
  • fix for a bug in faxmail(1) handling MIME; submitted by zorzella@dcc.unicamp.br (Luiz Otavio Lautenschlaeger Zorzella), 8 Jan 1997
  • fix for problem with xferfaxstats(1M) for values greater than 99 hours; submitted by david@southern.com (David), 7 Mar 1997
  • added -a option to faxrm(1) to allow administrator to remove other users jobs; submitted by jonc@pinnacle.co.nz (Jonathan Chen), 24 Sep 1997
  • fixed bug in hfaxd(1M) if "gethostbyname(hostname)" returns NULL for some reasons; submitted by dgilbert@jaywon.pci.on.ca (David Gilbert), 13 May 1997
  • Under Linux (RedHat 4.1), running NIS, the hostname/address resolution fails to properly resolve a given hostname. It appears to be a bug somewhere in the Linux. submitted by mvitek@emccta.com (Mark Vitek), 16 Jul 1997
  • fix to get faxsetup(1M) installed when doing a make installClient and fix for -client command line option; submitted by tim@trr.metro.net (Tim Rice), 25 Feb 1997
  • added a call to usermod(1M) command to faxsetup.linux to use the correct syntax on S.u.S.E. Linux distribution;
    NB: now faxsetup(1M) should set the correct uid for the user "fax" on Linux systems;
  • fix of a typo in faxsetup(1M) for making links to bin/ps2fax if ln(1) command has no "-s" flag;
  • NEW: mailfax script for qmail; submitted by Morten.Skjelland@chembio.ntnu.no (Morten Skjelland), 23 Jul 1997
  • removed "*F0" from the zyxel-2864 config-file (it's really a firmware bug)
  • patch used to build the SCO OS5 binary; submitted by tim@trr.metro.net (Tim Rice), 12/1996
  • patch for HylaFAX 4.0pl1 running on UnixWare 2.03 and 2.1.2 UnixWare's version of priocntl(2); submitted by tim@trr.metro.net (Tim Rice), 22 Sep. 1997
  • patch to convert network byte order into host byte order, patch to use "%f" instead of "%g" for precision by reading the typerules(4F) file; submitted by jik@kamens.brookline.ma.us (Jonathan I. Kamens), 18 Jul 1997
  • fix for ignored time specification after day of the week in the destctrls(4F) file; submitted by stuart@spook.ext.infobiogen.fr (Stuart Pook), 24 Aug 1997
  • fix a bug which SIGSEGV's the hfaxd(1M) if no password is given but the user can't use the service without password; submitted by thoth@purplefrog.com (Bob Forsman), 9 Oct 1997
  • new modem config file for SIERRA chip-set modem (e.g. Creatix); submitted by achimg@cologne.sgi.com (Achim Gorski), 19 Jul 1997
  • fix to correct wrap around the sequence file numbers is the queues; submitted by Philip.Abercrombie@MCS.VUW.AC.NZ (Phil Abercrombie), 25 Sep 1997
  • added a fontmap.pl script; submitted by evan@telly.org (Evan Leibovitch), 3 Oct 1997
  • corrected the man page for config(4F) for default of MaxDials: to 12; submitted by tlb@viaweb.com (Trevor Blackwell), Oct 1997
  • fix for a racing between faxsend(1M)'s renaming of documents and the faxq(1M)'s cleaning-up in the docq; submitted by matthias, 26 Oct 1997; see also hints from I.Senior@bom.gov.au (Ian Senior) and john@BlueSkyTours.COM (John Patrick Poet)
  • Compiler/portability fix (args to printf) in choptest(1M); submitted by sam@sgi.com (Sam Leffler), 24 Nov 1997
  • corrected a Base class destructor in faxmail(1) to be virtual. submitted by sam@sgi.com (Sam Leffler), 24 Nov 1997
  • Boolean params in config(4F) are now caseless; submitted by sam@sgi.com (Sam Leffler), 24 Nov 1997
  • Compiler/portability fix (arg to printf) for reading typerules(4F) file; submitted by sam@sgi.com (Sam Leffler), 24 Nov 1997
  • Notification, priority, and MST params are now caseless; submitted by sam@sgi.com (Sam Leffler), 24 Nov 1997
  • Notification params are now caseless. submitted by sam@sgi.com (Sam Leffler), 24 Nov 1997
  • new modem descriptions for K56flex Modems, with new modem descriptions for K56flex Modems, Multi-Tech MT2834ZDX and Hayes Accura 288 by Philip.Abercrombie@MCS.VUW.AC.NZ (Phil Abercrombie) and submiited in Nov 1997;
  • patch/addons which allow SVR4 machines to do a "make package" to produce packages that can be installed with the pkgadd(1M) command. It makes a client only package and a full package. submitted by tim@trr.metro.net (Tim Rice), 13 Nov 1997
  • added the filename to syslog message if the dialrules(4F) can't be opened
  • fix to make ModemMinSpeed: from config(4F) file working; submitted by hahne@aisol.net (Bruce M. Hahne), Nov 24, 97
  • UCP support for pagesend(1M) was added by afm@othello.ch (Andreas Mueller), Jan 5, 98; see also ixotap.html and the man pages of info(4F) pagesend(1M) and sendpage(1) for more information;
  • removed declared but never referenced "fxStr emsg;" in faxd/pageSendApp.c++;
  • HOWTO.html added to the pages installed during make install and HOWTO.html contains a pointer to the new ftp site;
  • small fix for situation "nothing to purge older than $AGEINFO days:" suggested by seth@psy.au.dk (Seth Chaiklin), 15 Jan 1998
  • added CONFIG_OPENFIFO=O_RDWR for FreeBSD systems with release 3.0; suggested by lars@fredriks-1.pr.mcs.net (Lars Fredriksen), January 30, 1998
  • changed the comments in faxmail/mailfax.sh-smail and faxmail/mailfax.sh-sendmail regarding DNS lookup; suggested by david@djwhome.demon.co.uk (David Woolley), 8 Feb 1998
  • added echo "--$MIMEBOUNDARY" ... to util/faxrcvd.sh sugested by benedix@rzaix530.rz.uni-leipzig.de (Ulf Benedix), Feb 11, 98
