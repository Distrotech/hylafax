HYLAFAX CHANGE INFORMATION
    Current Version: v4.0beta013
    Previous Version: v4.0beta012
    Master FTP Site: sgi.com (192.48.153.1), directory sgi/fax

Note that this is a major new distribution and lots of things have changed. Consult the general overview for a high-level description of the new functionality provided in the v4.0 software.

This document describes the changes made to the software between the previous and current versions (see above). If you don't find something listed here, then it was not done in this timeframe, or it was not considered important enough to be mentioned. The following information is located here:

  • Changes in the software configuration
  • Changes to hfaxd
  • Changes to faxsetup
  • Changes to faxmail
  • Changes in miscellaneous stuff

CHANGES IN THE SOFTWARE CONFIGURATION:
  • various problems associated with rereading parameters written to the config.cache file were corrected

CHANGES TO HFAXD:
  • hfaxd now becomes the super-user before doing a chown system call to deal with BSD-based systems

CHANGES TO FAXSETUP:
  • quoted file parameters are now properly handled when checking that the file exists
  • the logic for finding a process on a BSD-derived system was improved
  • references to the SBIN directory were corrected

CHANGES TO FAXMAIL:
  • a bug that caused the envelope to not include the input text was fixed

CHANGES IN MISCELLANEOUS STUFF:
  • faxcron works on systems that don't have any old-style log files
