MODEM BUGS AND GOTCHAS
This document contains information about modem bugs and gotchas that have been gleaned from experiences making the FlexFAX/HylaFAX code work with the modems. These problems are based either on my direct experiences or on the experiences of others that have worked with the software. Every attempt has been made to verify that these problems exist and are reproducible. I also make an effort to follow through with vendors when I encounter problems.

One should always check the version of the modem firmware. Note also that if a modem is not listed here then I have not tried it and/or noone has supplied a reliable report on the modem.
