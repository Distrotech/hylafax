MODEM BUGS AND GOTCHAS
This document contains information about modem bugs and gotchas that have been gleaned from experiences making the FlexFAX/HylaFAX code work with the modems. These problems are based either on Sam's direct experiences or on the experiences of others that have worked with the software. Every attempt has been made to verify that these problems exist and are reproducible.

One should always check the version of the modem firmware.
