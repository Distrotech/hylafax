CLASS 2 SUPPORT

Initial versions excluded Class 2 support. It is not known at what firmware version Class 2 support was re-introduced

  • ATI6 RCV56DPF-PLL L8571A Rev 19.00/19.00

  • ATI6 RCV56DPF L8570A Rev 47.24/47.24
