MULTI-TECH MT1432BA
    Manufacturer: Multi-Tech Systems
    Model: MT1432BA
    Latest Firmware Revision Tried: 1.12
    Recommended Firmware Revisions: 1.09 and later
    Last Updated:$Date$
There are several different models, of the MT1432, all of which function well. Models based on the AT&T datapump have firmware revisions numbered ``01xx'', while older models based on the Rockwell datapump have firmware revisions numbered ``03xx''.
This modem is recommended for use with HylaFAX.
CLASS 2 SUPPORT
  • Problems were reported communicating with Sharp model 6200 fax machines (fixed in rev 1.09 firmware)
  • The EC (error correction mode) subparameter reported in +FDIS: result strings can contain an invalid value (2)
