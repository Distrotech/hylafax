CLASS 2 SUPPORT
  • Zero-padding in the EOL codes of received facsimile data is often not zero when the Phase B+D bit ordering is set to "direct"; a workaround is to use AT+FBOR=2 and do bit reversal in the host
  • Problems have been observed receiving 2D-encoded data; if you have problems, disable its use through the AT+FDCC command.
