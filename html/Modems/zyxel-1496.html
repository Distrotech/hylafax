CLASS 2 SUPPORT

Most of the problems/issues found were fixed in the rev 6.13 firmware.

CLASS 2.0 SUPPORT
  • Several comments have been made to the mailing-list to say that this modem suffers particularly from the retrain negative problem [ will send the first page three times, then abort ].
  • However, the latest firmware, v6.19, has been reported to solve this issue; and the modem can be recommended for use with HylaFAX with this firmware revision or later.
  • Problems were reported with the 1496EG+ model available in the German market. The advice was to get an old dusty 1496E.
  • A 99.8% error free rate was the result of lowering the baud rate to 19200 in both the ModemRate as well as the S18 register. These are the necessary additions to the modem config file: 
    # Additional reset commands:
#
# &B1         DTE-DCE rate is fixed at DTE setting
# &N0         Auto-negotiate highest possible DCE-DCE link rate
# &S0         DSR always on
# *F0         Deny remote configuration
# S18=3       Receive at 19200
# S38.3=1     DCD on/off sequence follows UNIX standard; also
#             fix receiving baud rate at S18 value
# S39=0       (avoid Class 2 compatibility hacks)
#
ModemResetCmds:      AT&B1&N0&S0*F0S18=3S38.3=1S39=0
#
